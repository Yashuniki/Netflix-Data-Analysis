Netflix Data Analysis Project
 Project Overview

This project analyzes Netflix's movies and TV shows dataset to uncover valuable insights about content distribution, genres, ratings, countries, release trends, and more. Using Python and data visualization libraries, we explore patterns in Netflix's content library and present findings through charts and graphs.

Objectives
Analyze Netflix content data.
Identify trends in movies and TV shows.
Explore content ratings and genres.
Find top countries producing Netflix content.
Visualize release year trends.
Generate meaningful business insights.
Dataset

The dataset contains information about Netflix titles, including:
Title
Type (Movie/TV Show)
Director
Cast
Country
Release Year
Rating
Duration
Genre
Date Added

Dataset Source: Netflix Titles Dataset from Kaggle.

Python

1. Data Cleaning

Key Insights

Movies make up the majority of Netflix content.
The United States contributes the highest number of titles.
Netflix content significantly increased after 2015.
Drama and International Movies are among the most popular genres.
TV-MA is one of the most common content ratings.
